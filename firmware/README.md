# Firmware

This folder contains the code used for the smart glove project.

## Content

- flex sensor test programs
- IMU test programs
- integrated glove system code
- gesture demonstration code

## Purpose

The firmware is used to:
- read the five flex sensors
- process finger bending states
- read IMU data
- combine sensor inputs into one system
- test and recognize gestures

## Notes

The project was developed progressively, starting from sensor testing and calibration, then moving to integrated signal processing and finally to gesture implementation.
